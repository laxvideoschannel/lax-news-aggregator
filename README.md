# LaxHub

## Admin video backend

The site now includes:

- `GET /api/videos`
  - returns official PLL/WLL YouTube feed items plus any custom videos stored in Supabase
- `POST /api/admin/videos`
  - admin-only endpoint for adding custom YouTube videos from any channel
  - regular users do not have a frontend form for this

### Required environment variables

```env
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...
ADMIN_VIDEO_API_KEY=your-secret-admin-key
```

### Supabase table

Create this table in Supabase:

```sql
create table if not exists public.lacrosse_videos (
  id bigint generated by default as identity primary key,
  title text not null,
  youtube_url text not null unique,
  channel_name text,
  league text not null default 'CUSTOM',
  description text,
  featured boolean not null default false,
  thumbnail_url text,
  published_at timestamptz not null default now(),
  created_at timestamptz not null default now()
);
```

### Add a custom video

Example request:

```bash
curl -X POST https://your-site.com/api/admin/videos \
  -H "Content-Type: application/json" \
  -H "x-admin-key: your-secret-admin-key" \
  -d '{
    "title": "Blaze Riorden film breakdown",
    "youtubeUrl": "https://www.youtube.com/watch?v=VIDEO_ID",
    "league": "PLL",
    "channelName": "Your Channel Name",
    "description": "Optional custom description",
    "featured": true
  }'
```

Accepted `league` values:

- `PLL`
- `WLL`
- `CUSTOM`
